Formato do Arquivo CSV

Para que o arquivo CSV seja processado corretamente, ele deve seguir as especificações abaixo:

  • Formato: O arquivo deve estar no formato CSV (Comma-Separated Values).
  • Codificação: O arquivo deve estar codificado em UTF-8 para evitar problemas com caracteres especiais.
  • Separador: O separador de valores deve ser uma vírgula (,).
  • Cabeçalho: O arquivo deve conter as seguintes colunas no cabeçalho:
    • department: Nome do departamento.
    • title: Título ou cargo.
    • access: Tipo de acesso.
    • count: Quantidade de ocorrências.
  • Exemplo de Cabeçalho: 
    name,department,title,system,access
  • Exemplo de Linha: 
    Jess,Finance,Manager,App01,Full Access
  • Outros Requisitos:
    • Certifique-se de que não há linhas em branco no arquivo.
    • Os valores na coluna count devem ser numéricos.
